The base file of this template is identified as: "00_article-sbsr.tex";

The files that comprise the paper are named with numbers "01_ .." to "07_ ..";

Information about the references should be inserted in the file "07_references-sbsr.bib" as bibtex format;

Observations regarding the pagination and arrangement of figures are presented in "extra_considerations.tex". This section should be disabled for the document compiling;

The other files that come in this model have the global settings and should NOT be modified.
